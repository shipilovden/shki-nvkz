## Настройка темы

Настройка дизайна Magic Portfolio основана на конфигурации Once UI через data-атрибуты. Вы можете создать пользовательскую цветовую палитру для брендовых, акцентных и нейтральных цветов на [Once UI](https://once-ui.com/customize), где найдете инструкции по применению.

<CodeBlock
    marginBottom="16"
    codes={[
  {
    code:
`theme:       'dark',         // dark | light
neutral:     'gray',         // sand | gray | slate
brand:       'blue',         // blue | indigo | violet | magenta | pink | red | orange | yellow | moss | green | emerald | aqua | cyan
accent:      'indigo',       // blue | indigo | violet | magenta | pink | red | orange | yellow | moss | green | emerald | aqua | cyan
solid:       'contrast',     // color | contrast

border:      'playful',      // rounded | playful | conservative

transition:  'all',          // all | micro | macro
scaling:     '100',          // 90 | 95 | 100 | 105 | 110`, 
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

## Визуальные эффекты

В `layout.tsx` есть предварительно настроенный фон, который вы можете изменить в конфигурационном файле. Настройте графические элементы, такие как градиент, точки, линии и сетку, и настройте их внешний вид.

## Кастомизация компонентов

Once UI предоставляет множество компонентов, которые можно легко настроить под ваши потребности. Каждый компонент имеет встроенные варианты стилей и может быть дополнительно настроен через CSS-переменные.

## Адаптивность

Все компоненты Once UI полностью адаптивны и автоматически подстраиваются под различные размеры экранов. Это обеспечивает отличный пользовательский опыт на всех устройствах.

## Производительность

Once UI оптимизирован для производительности и включает в себя только необходимые стили, что обеспечивает быструю загрузку вашего портфолио.
