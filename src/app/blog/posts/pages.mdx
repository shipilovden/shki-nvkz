## Включение или отключение страниц

Компонент `RouteGuard` в Magic Portfolio отвечает за условный рендеринг страниц на основе объекта `routes` в файле `resources/config.js`.

<CodeBlock
    marginBottom="16"
    highlight="6"
    codes={[
  {
    code:
`const routes = {
    '/':        true,
    '/about':   true,
    '/work':    true,
    '/blog':    true,
    '/gallery': false,
};`,
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

Код выше обеспечит, что страница `/gallery` будет недоступна и не будет отображаться в навигации.

## Добавление новых страниц

При создании новой страницы её необходимо добавить в объект `routes` в файле `resources/config.js`.

<CodeBlock
    marginBottom="16"
    highlight="6"
    codes={[
  {
    code:
`const routes = {
    '/':        true,
    '/about':   true,
    '/work':    true,
    '/blog':    true,
    '/gallery': false,
    '/music': true,
};`,
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

Код выше обеспечит, что страница `/music` будет доступна. Пользователи смогут перейти к ней после добавления её в меню навигации в файле `components/Header.tsx`.
