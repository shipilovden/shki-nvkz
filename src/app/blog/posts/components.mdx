Magic Portfolio заменяет некоторые стандартные HTML-элементы на компоненты Once UI для лучшей интеграции в дизайн и добавления дополнительной функциональности. Кроме того, многие распространенные компоненты Once UI уже импортированы и доступны для использования.

## Обратная связь

Компонент обратной связи используется для отображения критически важной информации посетителям.

## Код

Используйте компонент CodeBlock для отображения фрагментов кода с подсветкой синтаксиса, предварительным просмотром и функциональностью копирования в буфер обмена. Используйте компонент InlineCode для встроенных фрагментов кода.

### CodeBlock

Компонент блока кода работает на основе CodeBlock от Once UI. Вы можете получить к нему доступ с полным синтаксисом для полной настройки или используя стандартный синтаксис markdown (тройные обратные кавычки) с предварительно настроенными свойствами.

<Column fillWidth gap="-1" marginBottom="16" marginTop="8">
<CodeBlock
    radius="none"
    topRadius="l"
    codes={[
      {
        code:
`// Full component syntax
<CodeBlock
    codes={[
      {
        code: "console.log('Hello, World!');",
        language: "javascript",
        label: "Example"
      }
    ]}
/>

// Short syntax example
\`\`\`tsx
function greeting(name) {
    return \`Hello, \${name}!\`;
}

console.log(greeting('World'));
\`\`\``,
        language: "tsx",
        label: "CodeBlock"
      }
    ]}
/>
<Row fillWidth bottomRadius="l" padding="4" border="neutral-medium" background="surface">
    <Button href="https://once-ui.com/docs/codeBlock" weight="default" label="View on Once UI" size="s" variant="tertiary" suffixIcon="chevronRight"/>
</Row>
</Column>

### InlineCode

`InlineCode` — это еще один компонент Once UI, к которому вы можете получить доступ либо через полный синтаксис, либо через короткий синтаксис (обратные кавычки). Обычно рекомендуется стандартный синтаксис markdown, поскольку у него не так много вариантов конфигурации.

<Column fillWidth gap="-1" marginBottom="16" marginTop="8">
<CodeBlock
    radius="none"
    topRadius="l"
    codeHeight={24}
    preview={
    <InlineCode>
        const x = 10;
    </InlineCode>}
    codes={[
      {
        code:
`// Full component syntax
<InlineCode>
    const x = 10;
</InlineCode>

// Short syntax example
\`const x = 10;\``,
        language: "tsx",
        label: "InlineCode"
      }
    ]}
/>
<Row fillWidth bottomRadius="l" padding="4" border="neutral-medium" background="surface">
    <Button href="https://once-ui.com/docs/inlineCode" weight="default" label="View on Once UI" size="s" variant="tertiary" suffixIcon="chevronRight"/>
</Row>
</Column>

## Пользовательские компоненты

Поскольку Magic Portfolio основан на Once UI, вы можете использовать все компоненты, предоставляемые Once UI в файлах `.mdx`, но вам нужно сначала импортировать их в файл `src/product/mdx.tsx` и передать в объект `components`.

<CodeBlock
    marginBottom="16" marginTop="8"
    highlight="2"
    codes={[
      {
        code:
`const components = {
    // Add new components
    p: createParagraph as any,
    h1: createHeading("h1") as any,
    ...
    Media,
    SmartLink,
};`,
        language: "tsx",
        label: "src/product/mdx.tsx"
      }
    ]}
/>
